# 📚 Smart Study Buddy

Smart Study Buddy is a simple AI-powered study assistant built with **Streamlit** and the **Groq API**. It helps students understand study topics, programming concepts, and general questions through easy-to-understand AI-generated responses.

## 🚀 Features

- AI-powered question answering
- Beginner-friendly explanations
- Simple and clean user interface
- Fast responses using Groq API
- Built with Python and Streamlit

## 🛠️ Technologies Used

- Python
- Streamlit
- Groq API
- python-dotenv

## 📂 Project Structure

```
Smart-Study-Buddy/
│── app.py
│── .env
│── requirements.txt
└── README.md
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to the project folder

```bash
cd Smart-Study-Buddy
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your Groq API key

```env
GROQ_API_KEY=your_api_key_here
```

5. Run the application

```bash
streamlit run app.py
```

## 💡 How to Use

1. Open the application in your browser.
2. Type your study-related question.
3. Click **Get Answer**.
4. Read the AI-generated explanation.
